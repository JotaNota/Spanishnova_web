Practice: Real Conditional Sentences in Spanish

Oraciones afirmativas

  • Si hago ejercicio, me siento bien. – If I exercise, I feel good.
  • Si ella corre mucho, se cansa rápido. – If she runs a lot, she gets tired fast.
  • Si comemos temprano, nos dormimos mejor. – If we eat early, we sleep better.
  • Si ellos estudian, se sienten seguros. – If they study, they feel confident.

Oraciones negativas

  • Si no duermo bien, me siento mal. – If I don’t sleep well, I feel bad.
  • Si ella no se estira, se lastima. – If she doesn’t stretch, she gets hurt.
  • Si no practicamos, no mejoramos. – If we don’t practice, we don’t improve.

Oraciones interrogativas

  • Si te acuestas tarde, ¿te levantas cansado? – If you go to bed late, do you wake up tired?
  • Si ella se apura, ¿se equivoca? – If she rushes, does she make mistakes?
  • Si hacen yoga, ¿se relajan? – If they do yoga, do they relax?
  • Si te enojas, ¿por qué te callas? – If you get angry, why do you go quiet?

Traducir

1. If I eat vegetables, I feel good. →

Si como verduras, me siento bien.

2. If you forget the password, you get worried. →

Si olvidas la contraseña, te preocupas.

3. If she drinks coffee late, she doesn’t sleep. →

Si ella bebe café tarde, no duerme.

4. If he stretches, he relaxes. →

Si él se estira, se relaja.

5. If we leave early, we arrive on time. →

Si salimos temprano, llegamos a tiempo.

6. If they don’t practice, they don’t improve. →

Si no practican, no mejoran.

7. If I see the mistake, I fix it. →

Si veo el error, lo arreglo.

8. If you sit here, you hear better. →

Si te sientas aquí, oyes (escuchas) mejor.

9. If she gets dressed quickly, she saves time. →

Si ella se viste rápido, ahorra tiempo.

10. If he brings snacks, everyone is happy. →

Si él trae meriendas, todos están felices.

11. If we remember the rule, we don’t make mistakes. →

Si recordamos la regla, no cometemos errores.

12. If they lie down, they rest. →

Si se acuestan, descansan.

13. If I start late, I finish late. →

Si empiezo tarde, termino tarde.

14. If you pay attention, you understand. →

Si prestas atención, entiendes.

Andrea entrena con cuidado

Andrea va al gimnasio cada mañana. Ella hace los ejercicios con calma y paciencia. Si ella se apura, se equivoca con los ejercicios. Ella prefiere ir despacio porque si hace un movimiento rápido, se lastima.

En el gimnasio, Andrea entrena con su amiga Lucía. Si hay mucho ruido, ellas se callan para concentrarse.

A veces los ejercicios son difíciles, pero Andrea no se enoja. Si ella se siente cansada, descansa un poco y luego sigue con su rutina.

Si Andrea termina su entrenamiento, ella y su amiga toman un batido de proteína. Su favorito es de chocolate.

Ella sabe que si sigue su plan, se siente fuerte y feliz.
