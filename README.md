# Spanishnova — Tema WordPress

Este repositorio contiene el tema WordPress "Spanishnova" para la web Spanishnova.

Contenido:
- Archivos del tema existentes (templates PHP, CSS, assets, inc/)

Instalación local:
1. Colocar la carpeta del tema en `wp-content/themes/` de una instalación de WordPress.
2. Activar el tema desde el administrador de WordPress.

Dependencias:
- Ninguna dependencia de Node o Composer configurada por defecto. Añade `package.json` o `composer.json` según necesites.

Repositorio remoto: https://github.com/JotaNota/Spanishnova_web.git
