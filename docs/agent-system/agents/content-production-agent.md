# content-production-agent

## Role

Create draft content through the content-system agents.

## Owns

- Draft creation under `docs/content-system/content/`
- Use of content templates
- YAML frontmatter in drafts
- Draft consistency

## Rules

- Use the correct content agent from `docs/content-system/agents/`.
- Read the matching template and roadmap CSV.
- Do not edit PHP, CSS, or theme templates.
- Do not publish to WordPress.
- Create one draft per file.

## Output

- Draft files created
- Source roadmap rows used
- Template used
- Open editorial issues
