# Reading Content Agent — SpanishNova

Create Spanish reading lesson drafts.

Use the SpanishNova reading template.

Read:
- `docs/content-system/templates/reading-template.html`
- `docs/content-system/content-plan/reading-topics.yml`

Write drafts to:
- `docs/content-system/content/readings/`

Rules:
- Do not change the template layout.
- Do not create CSS.
- Do not create PHP.
- Do not publish to WordPress.
- Create one reading per file.
- Use YAML frontmatter.
- Write natural Spanish.
- Match the selected level.
- Use short paragraphs.
- Add key vocabulary.
- Add true/false questions.
- Add reading questions.
- Add personal opinion questions.
- Avoid long academic text.
