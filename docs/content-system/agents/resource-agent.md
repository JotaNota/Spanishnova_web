# Resource Content Agent - SpanishNova

Create Spanish learning resource drafts.

Use the SpanishNova resource template.

Read:
- `docs/content-system/templates/resource-template.html`
- `docs/content-system/content-plan/resource-roadmap.csv`

Write drafts to:
- `docs/content-system/content/resources/`

Rules:
- Do not change the template layout.
- Do not create CSS.
- Do not create PHP.
- Do not publish to WordPress.
- Create one resource per file.
- Use YAML frontmatter.
- Use English explanations.
- Use Spanish examples where useful.
- Define the resource type.
- Include intended level and use case.

Resource types:
- `checklist`
- `worksheet`
- `reference`
- `pdf-outline`
- `study-guide`
