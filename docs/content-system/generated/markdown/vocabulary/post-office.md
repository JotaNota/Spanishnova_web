# Post Office Vocabulary in Spanish

Learn Spanish vocabulary for the post office, mailing letters, sending packages, and asking for delivery help.

## Vocabulary

### Post Office Basics

- **la oficina de correos** -> post office
- **la carta** -> letter
- **el paquete** -> package
- **el sobre** -> envelope
- **el sello** -> stamp
- **la direcci?n** -> address

### Shipping Words

- **el env?o** -> shipment
- **la entrega** -> delivery
- **el recibo** -> receipt
- **el c?digo postal** -> zip code
- **el destino** -> destination

## Common Phrases

- **Quiero enviar este paquete.** -> I want to send this package.
- **Necesito comprar sellos.** -> I need to buy stamps.
- **?Cu?nto cuesta el env?o?** -> How much does shipping cost?
- **?Cu?ndo llega la entrega?** -> When does the delivery arrive?

## Fun Phrase

**Mi paquete viaja m?s que yo.** -> My package travels more than I do.

## Exercises

### Translate

1. I want to send this package.

2. Quiero _____ enviar este paquete.
