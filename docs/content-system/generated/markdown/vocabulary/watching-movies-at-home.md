# Watching Movies at Home Vocabulary in Spanish

Learn Spanish vocabulary for watching movies at home, choosing what to watch, and talking about snacks and opinions.

## Vocabulary

### Movie Night Basics

- **la pel?cula** -> movie
- **la serie** -> series
- **la pantalla** -> screen
- **el control remoto** -> remote control
- **los subt?tulos** -> subtitles
- **el sof?** -> sofa

### Genres and Snacks

- **la comedia** -> comedy
- **el drama** -> drama
- **el terror** -> horror
- **las palomitas** -> popcorn
- **la bebida** -> drink
- **recomendar** -> to recommend

## Common Phrases

- **Vamos a ver una pel?cula en casa.** -> We are going to watch a movie at home.
- **?Quieres subt?tulos?** -> Do you want subtitles?
- **Traigo palomitas.** -> I am bringing popcorn.
- **Esta pel?cula es muy divertida.** -> This movie is very funny.

## Fun Phrase

**Elegir la pel?cula tarda m?s que verla.** -> Choosing the movie takes longer than watching it.
