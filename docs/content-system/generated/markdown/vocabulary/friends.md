# Friends Vocabulary in Spanish

Learn Spanish vocabulary for talking about friends, plans, invitations, and social life.

## Vocabulary

### People

- **el amigo** -> friend
- **la amiga** -> friend
- **el mejor amigo** -> best friend
- **la amistad** -> friendship
- **el grupo** -> group

### Plans

- **salir** -> to go out
- **quedar** -> to meet up
- **invitar** -> to invite
- **charlar** -> to chat
- **celebrar** -> to celebrate

## Common Phrases

- **Voy a salir con mis amigos.** -> I am going out with my friends.
- **?Quieres quedar esta tarde?** -> Do you want to meet up this afternoon?
- **Mi mejor amiga es muy divertida.** -> My best friend is very fun.
- **Tenemos un plan para el s?bado.** -> We have a plan for Saturday.

## Fun Phrase

**Un buen amigo siempre trae buena energ?a.** -> A good friend always brings good energy.
