Indirect object pronouns show who receives, benefits from, or is affected by an action. They often answer to whom or for whom.

## Overview

The main indirect object pronouns are me, te, le, nos, and les. They usually appear before the conjugated verb: me escribe, le doy, nos mandan.

- Le escribo a Ana. -> I write to Ana.
- Nos dieron la información. -> They gave us the information.

## Cómo/cuándo lo usamos

### Communication and giving

Indirect objects are common with decir, escribir, mandar, dar, explicar, and contar.

- Te mandé el archivo. -> I sent you the file.
- Les expliqué la regla. -> I explained the rule to them.

### Clarify le and les

Le and les can refer to different people, so Spanish often adds a + person for clarity.

- Le escribí a mi jefe. -> I wrote to my boss.
- Les di el número a mis padres. -> I gave the number to my parents.

## Estructura

| Patrón | Example | Translation |
| --- | --- | --- |
| indirect object pronoun + verb + thing | Le mando un mensaje. | I send him a message. |
| no + indirect object pronoun + verb | No me respondió. | She did not answer me. |
| indirect object pronoun + verb + a + person | Le escribo a Ana. | I write to Ana. |

## Oraciones.

### Afirmativas

- Le escribo a Ana. -> I write to Ana.
- Me dieron una respuesta. -> They gave me an answer.
- Nos mandaron el archivo. -> They sent us the file.
- Les expliqué el problema. -> I explained the problem to them.

### Negativas

- No le dije nada. -> I did not tell him anything.
- Ella no me respondió. -> She did not answer me.
- No nos enviaron la dirección. -> They did not send us the address.
- No les gustó la idea. -> They did not like the idea.

### Forma de pregunta

- ¿Me mandaste el enlace? -> Did you send me the link?
- ¿Le escribiste a tu jefe? -> Did you write to your boss?
- ¿Nos explican el plan? -> Do they explain the plan to us?
- ¿Les diste el número? -> Did you give them the number?
