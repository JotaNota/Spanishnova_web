Learn useful Spanish weather vocabulary for talking about the forecast, seasons, and everyday plans.

Vocabulary

Weather Conditions

  • el tiempo → weather
  • el sol → sun
  • la lluvia → rain
  • la nieve → snow
  • el viento → wind
  • la tormenta → storm

Descriptions

  • hace calor → it is hot
  • hace fr?o → it is cold
  • est? soleado → it is sunny
  • est? nublado → it is cloudy
  • llueve → it is raining
  • el pron?stico → forecast

Common Phrases

  • ?Qu? tiempo hace hoy? → What is the weather like today?
  • Hace mucho calor. → It is very hot.
  • Parece que va a llover. → It looks like it is going to rain.
  • Necesito un paraguas. → I need an umbrella.

Fun Phrase

No hay mal tiempo, solo ropa equivocada. → There is no bad weather, only the wrong clothes.
