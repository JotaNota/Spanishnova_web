Learn Spanish vocabulary for text messages, chats, notifications, and everyday digital communication.

Vocabulary

Message Words

  • el mensaje → message
  • el chat → chat
  • la notificaci?n → notification
  • el grupo → group
  • el emoji → emoji
  • la respuesta → reply

Actions

  • mandar → to send
  • recibir → to receive
  • responder → to reply
  • escribir → to write
  • borrar → to delete
  • reenviar → to forward

Common Phrases

  • Te mando un mensaje. → I will send you a message.
  • No vi tu mensaje. → I did not see your message.
  • Resp?ndeme cuando puedas. → Reply when you can.
  • Estoy en un chat de grupo. → I am in a group chat.

Fun Phrase

Escrib? un mensaje largo y solo respondieron con un emoji. → I wrote a long message and they only answered with an emoji.
