Overview

Reported speech lets you say what someone said, asked, or explained without quoting every word directly. Spanish often shifts tense, pronouns, and time expressions. Focus on the message first, then choose the form that fits the context.

  • Reported speech: ejemplo principal. → Reported speech: main example.
  • Reported speech: otro ejemplo ?til. → Reported speech: another useful example.

Cómo/cuándo lo usamos

Real use

Use this pattern when reported speech helps express a real message, not just a grammar label.

  • Reported speech: ejemplo pr?ctico uno. → Reported speech: practical example one.
  • Reported speech: ejemplo pr?ctico dos. → Reported speech: practical example two.

Common context

This form often appears in everyday explanations, plans, advice, and reactions depending on the sentence.

  • Reported speech: contexto com?n uno. → Reported speech: common context one.
  • Reported speech: contexto com?n dos. → Reported speech: common context two.

Estructura

Patrón Example Translation
trigger + que + verb form Reported speech: quiero que uses la forma. I want you to use the form.
si + condition, result Reported speech: si cambia el contexto, cambia la forma. If the context changes, the form changes.
reported/passive pattern Reported speech: la idea se expresa con claridad. The idea is expressed clearly.
