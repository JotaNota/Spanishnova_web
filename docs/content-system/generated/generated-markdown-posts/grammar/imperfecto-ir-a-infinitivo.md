Iba a + infinitive talks about plans, intentions, or expected actions in the past. It is the past version of voy a + infinitive.

## Overview

Use iba, ibas, iba, íbamos, or iban + a + infinitive when someone was going to do something.

- Iba a llamar, pero me quedé sin batería. -> I was going to call, but my battery died.
- Íbamos a salir, pero empezó a llover. -> We were going to go out, but it started to rain.

## Cómo/cuándo lo usamos

### Past plans

Use it for a plan someone had before another event or decision changed things.

- Iba a cocinar, pero pedimos comida. -> I was going to cook, but we ordered food.
- Ellos iban a viajar en julio. -> They were going to travel in July.

### Expected action

It can also show what was expected or about to happen.

- La clase iba a empezar a las ocho. -> The class was going to start at eight.
- Íbamos a entrar cuando llegó Ana. -> We were going to enter when Ana arrived.

## Estructura

| Patrón | Example | Translation |
| --- | --- | --- |
| subject + iba/ibas/íbamos/iban + a + infinitive | Yo iba a estudiar. | I was going to study. |
| no + iba a + infinitive | No iba a decir nada. | I was not going to say anything. |
| question + iba a + infinitive | ¿Ibas a venir? | Were you going to come? |

## Oraciones.

### Afirmativas

- Yo trabajaba en el centro. -> I used to work downtown.
- Tú comías con tu familia. -> You used to eat with your family.
- Ella vivía cerca. -> She used to live nearby.
- Nosotros íbamos al parque. -> We used to go to the park.

### Negativas

- Yo no salía mucho. -> I did not use to go out much.
- Él no tenía coche. -> He did not have a car.
- Nosotras no veíamos televisión. -> We did not use to watch TV.
- Ellos no eran vecinos. -> They were not neighbors.

### Forma de pregunta

- ¿Trabajabas los sábados? -> Did you use to work on Saturdays?
- ¿Ella vivía aquí? -> Did she use to live here?
- ¿Íbamos juntos? -> Did we use to go together?
- ¿Ustedes veían esa serie? -> Did you all use to watch that series?
