# School Vocabulary in Spanish

Learn Spanish school vocabulary for classrooms, supplies, subjects, and everyday student life.

## Vocabulary

### People and Places

- **la escuela** -> school
- **la clase** -> class
- **el aula** -> classroom
- **el profesor** -> teacher
- **la profesora** -> teacher
- **el estudiante** -> student

### Supplies and Subjects

- **el libro** -> book
- **el cuaderno** -> notebook
- **el l?piz** -> pencil
- **la mochila** -> backpack
- **la tarea** -> homework
- **matem?ticas** -> math

## Common Phrases

- **Tengo clase a las ocho.** -> I have class at eight.
- **Necesito mi cuaderno.** -> I need my notebook.
- **La tarea es para ma?ana.** -> The homework is for tomorrow.
- **El examen es dif?cil.** -> The test is difficult.

## Fun Phrase

**Mi mochila tiene de todo menos tiempo libre.** -> My backpack has everything except free time.
