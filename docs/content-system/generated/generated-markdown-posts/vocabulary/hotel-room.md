# La Habitación Del Hotel - Hotel Room

Learn useful Spanish words and phrases for checking into a hotel room, asking about room features, and solving small problems during a trip.

## Overview

This vocabulary helps you talk about the hotel room itself: the bed, key, bathroom, towels, and basic services. Use it when you arrive, ask for something, or explain what you need.

- Necesito otra toalla. - I need another towel.

## Vocabulary

### Room Items

| Spanish | English |
| --- | --- |
| la habitación | the room |
| la cama | the bed |
| la llave | the key |
| la ventana | the window |
| el baño | the bathroom |
| la ducha | the shower |

### Hotel Needs

| Spanish | English |
| --- | --- |
| una toalla | a towel |
| una almohada | a pillow |
| una manta | a blanket |
| el aire acondicionado | the air conditioning |
| la recepción | the front desk |
| el ascensor | the elevator |

## Common Phrases

1. Spanish: Necesito otra toalla.
   English: I need another towel.
2. Spanish: La llave no funciona.
   English: The key does not work.
3. Spanish: Hay un problema con la ducha.
   English: There is a problem with the shower.
4. Spanish: Quiero una habitación tranquila.
   English: I want a quiet room.

## Dialogue

### At The Front Desk

- Guest: Hola, la llave de mi habitación no funciona.
- Receptionist: Lo siento. ¿Cuál es su número de habitación?
- Guest: Es la habitación doscientos cuatro. También necesito otra toalla.
- Receptionist: Claro. Aquí tiene una llave nueva, y enviaremos una toalla.

## Exercises

### Translate

1. Prompt: I need another towel.
   Answer: Necesito otra toalla.
2. Prompt: The key does not work.
   Answer: La llave no funciona.
3. Prompt: I want a quiet room.
   Answer: Quiero una habitación tranquila.
4. Prompt: There is a problem with the shower.
   Answer: Hay un problema con la ducha.

### Write Or Speak

1. Ask the front desk for another pillow.
2. Explain that the air conditioning does not work.
3. Say that you want a room near the elevator.

## Questions

1. ¿Qué necesitas en tu habitación?
2. ¿Prefieres una habitación tranquila o una habitación cerca del ascensor?
3. ¿Qué dices si la llave no funciona?

## Wrap Up
