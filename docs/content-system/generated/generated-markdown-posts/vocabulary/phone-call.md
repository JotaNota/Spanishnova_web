# Phone Call Vocabulary in Spanish

Learn Spanish vocabulary and phrases for making phone calls, answering calls, and leaving messages.

## Vocabulary

### Phone Words

- **el tel?fono** -> phone
- **la llamada** -> call
- **el mensaje** -> message
- **el buz?n de voz** -> voicemail
- **el n?mero** -> number
- **la se?al** -> signal

### Call Actions

- **llamar** -> to call
- **contestar** -> to answer
- **colgar** -> to hang up
- **dejar un mensaje** -> to leave a message
- **devolver la llamada** -> to return the call

## Common Phrases

- **Te llamo m?s tarde.** -> I will call you later.
- **No tengo se?al.** -> I do not have signal.
- **?Puedes dejar un mensaje?** -> Can you leave a message?
- **Voy a devolver la llamada.** -> I am going to return the call.

## Fun Phrase

**Mi tel?fono suena justo cuando no puedo contestar.** -> My phone rings exactly when I cannot answer.
