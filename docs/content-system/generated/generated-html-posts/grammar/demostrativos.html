Demonstratives point to people or things. In Spanish, este, ese, and aquel help you say this, that, and that over there.

Overview

Spanish demonstratives agree with the noun in gender and number. Este is close to the speaker, ese is farther away or near the listener, and aquel points to something more distant.

  • Este libro es mío. → This book is mine.
  • Esa casa es bonita. → That house is pretty.
  • Aquellos estudiantes son nuevos. → Those students over there are new.

Formas

  • este / esta / estos / estas → this / these
  • ese / esa / esos / esas → that / those
  • aquel / aquella / aquellos / aquellas → that over there / those over there

Notas

Demonstratives agree with the noun in gender and number.

Este points to something close, ese points to something farther away, and aquel points to something more distant.

Conjugación

Subject Forma Example Translation

Farther masculine ese / esos ese coche / esos coches that car / those cars
Farther feminine esa / esas esa casa / esas casas that house / those houses
Distant aquel / aquella / aquellos / aquellas aquellos edificios those buildings over there

Cómo/cuándo lo usamos

Close Things

Use este, esta, estos, and estas for things close to the speaker.

  • Esta silla es cómoda. → This chair is comfortable.
  • Estos zapatos son nuevos. → These shoes are new.

Farther Things

Use ese, esa, esos, and esas for things farther away or near the listener.

  • Ese restaurante es bueno. → That restaurant is good.
  • Esas preguntas son fáciles. → Those questions are easy.

Distant Things

Use aquel forms for things that feel farther away in space, time, or context.

  • Aquel edificio es antiguo. → That building over there is old.
  • Aquellas vacaciones fueron perfectas. → Those vacations back then were perfect.

Oraciones.

Afirmativas

  • Este café está frío. → This coffee is cold.
  • Esta clase es interesante. → This class is interesting.
  • Esos libros son caros. → Those books are expensive.
  • Aquella ciudad es famosa. → That city over there is famous.

Negativas

  • Este teléfono no funciona. → This phone does not work.
  • Esas ventanas no están abiertas. → Those windows are not open.
  • Aquel hotel no es barato. → That hotel over there is not cheap.
  • Estos ejercicios no son difíciles. → These exercises are not difficult.

Forma de pregunta

  • ¿Este libro es tuyo? → Is this book yours?
  • ¿Esa mochila es de Ana? → Is that backpack Ana's?
  • ¿Aquellos hombres son profesores? → Are those men over there teachers?
  • ¿Estas llaves son tuyas? → Are these keys yours?
