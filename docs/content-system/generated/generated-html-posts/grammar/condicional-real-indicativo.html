Real conditional sentences describe situations that are possible or realistic. In Spanish, many real conditionals use si with the indicative, followed by another indicative verb.

Overview

Use this structure when the condition can really happen. The si clause gives the condition, and the main clause gives the result.

  • Si tengo tiempo, estudio español. → If I have time, I study Spanish.
  • Si llueve, nos quedamos en casa. → If it rains, we stay at home.

Cómo/cuándo lo usamos

Real possibilities

Use the indicative when the condition is possible, factual, or likely.

  • Si Ana está en casa, la visitamos. → If Ana is at home, we visit her.
  • Si tú puedes, empezamos ahora. → If you can, we start now.

General truths and routines

Real conditionals can describe what normally happens when a condition is true.

  • Si trabajo mucho, me canso. → If I work a lot, I get tired.
  • Si los niños tienen hambre, comen fruta. → If the children are hungry, they eat fruit.

Estructura

Patrón Example Translation
si + present indicative, present indicative Si tengo hambre, como algo. If I am hungry, I eat something.
present indicative + si + present indicative Voy al parque si hace sol. I go to the park if it is sunny.
si + present indicative, future or command-like result Si necesitas ayuda, te llamo. If you need help, I will call you.

Oraciones.

Afirmativas

  • Si estudio, aprendo más. → If I study, I learn more.
  • Si tienes café, tomo una taza. → If you have coffee, I drink a cup.
  • Vamos al cine si termina temprano. → We go to the movies if it ends early.
  • Si hace frío, cierro la ventana. → If it is cold, I close the window.

Negativas

  • Si no tengo dinero, no compro nada. → If I do not have money, I do not buy anything.
  • No salimos si llueve. → We do not go out if it rains.
  • Si él no llama, no esperamos. → If he does not call, we do not wait.
  • Si no entiendes, no sigues adelante. → If you do not understand, you do not continue.

Forma de pregunta

  • ¿Qué haces si tienes tiempo? → What do you do if you have time?
  • ¿Vienes si te invito? → Do you come if I invite you?
  • ¿Si llueve, cancelamos la clase? → If it rains, do we cancel the class?
  • ¿Me llamas si necesitas ayuda? → Do you call me if you need help?
