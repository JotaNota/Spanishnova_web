# Conversation Structure Variants - SpanishNova

Use `conversation_variant` from `docs/content-system/content-plan/conversation-roadmap.csv` to choose the lesson shape.

## podcast_dialogue

Use for topic-based conversations with two recurring speakers.

Structure:
- Short English intro.
- Useful vocabulary.
- Main dialogue between two speakers.
- Inline English support only for selected useful lines.
- Common phrases from the dialogue.
- Practice section.
- Roleplay prompts.
- Discussion questions.

Content rules:
- Keep the dialogue natural.
- Use two speakers consistently.
- Let the conversation develop the topic instead of listing tips.
- Use enough turns to feel like a real podcast segment.

## street_interview

Use for interview-style lessons with one interviewer and several people.

Structure:
- Short English intro.
- Useful vocabulary.
- Interview blocks by person or group.
- Selected English support after useful questions or answers.
- Common phrases from the interviews.
- Practice section.
- Roleplay prompts.
- Discussion questions.

Content rules:
- Use one interviewer.
- Use several interviewees.
- Give each interviewee a distinct routine, problem, preference, or situation.
- Keep each block compact but complete.

## practical_roleplay

Use for functional situations learners may need in daily life.

Structure:
- Short English intro.
- Useful vocabulary.
- Common phrases.
- Main conversation.
- Practice section.
- Roleplay prompts.
- Discussion questions.
- Wrap up.

Content rules:
- Focus on one practical situation.
- Use clear speaker labels.
- Keep the language appropriate for the level.
- Include phrases learners can reuse.

## guided_exchange

Use for short controlled conversation practice.

Structure:
- Short English intro.
- Useful vocabulary.
- Model exchange.
- Variation prompts.
- Practice section.
- Roleplay prompts.
- Discussion questions.

Content rules:
- Keep the dialogue short.
- Prioritize repeatable sentence patterns.
- Use variations that help learners swap words or details.
