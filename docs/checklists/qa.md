# QA Checklist

Status: Draft scaffold

## WordPress

- Active theme confirmed.
- CPT archives load.
- Single templates load.
- Taxonomy archives load.
- Search works.

## Frontend

- Desktop check complete.
- Mobile check complete.
- Navigation works.
- Images load.
- No visible PHP warnings.

## SEO

- Pages are indexable where intended.
- One H1 per page.
- Titles and meta behavior confirmed.
- Internal links present.
- Sitemap behavior confirmed.

## Accessibility

- Keyboard navigation spot-check complete.
- Focus states visible.
- Color contrast spot-check complete.
- Images have useful alt text where needed.
