# Launch Checklist

Status: Draft scaffold

## Before launch

- Database backup complete.
- Files backup complete.
- Rollback path documented.
- Active theme confirmed.
- Required plugins confirmed.
- Cache behavior confirmed.
- Analytics confirmed.
- Search Console confirmed.

## Content

- Priority pages reviewed.
- Internal links reviewed.
- Broken links checked.
- SEO metadata reviewed.

## After launch

- Homepage checked.
- Key archives checked.
- Key single pages checked.
- Forms checked.
- Analytics events checked.
- Indexing monitored.
