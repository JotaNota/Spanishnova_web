# Guidance Schema

Status: Draft scaffold

Project guidance should define how humans and AI agents work on SpanishNova.

Required sections for project guidance:

- Role and intent
- Operating principles
- Execution protocol
- Content architecture rules
- Editorial workflow rules
- WordPress development constraints
- Verification and completion
- Recovery and lifecycle notes

Project-specific guidance can live in:

- Root project guidance files
- `docs/`
- `docs/content-system/`
- Agent or skill files when present

## Guidance rules

- Mark unknowns as `TBD`.
- Separate product assumptions from confirmed requirements.
- Keep editorial rules close to content system files.
- Keep development rules close to WordPress workflow files.
